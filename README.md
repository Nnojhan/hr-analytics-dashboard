# HR & Asset Analytics Dashboard

## Project Overview

This project demonstrates the development of HR analytics dashboards designed to support workforce monitoring.

## Objectives

- Monitor workforce demographics
- Analyze attendance and absenteeism
- Track overtime and business travel
- Support data-driven decision making

## Tools Used

- Microsoft Excel
- Google Sheets
- Looker Studio

## Dashboard Features

### HR Dashboard
- Employee Biography
- Attendance & Absenteeism
- Sick Leave Categories
- Overtime Analysis
- Business Travel
- Payroll & Overtime
- Employee Retention
- Workforce Growth
